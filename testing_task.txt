Создать консольное приложение для OS Linux, которое:

Обязательный блок:
- Может настраиваться через:
  - Конфигурационный файл yaml или json (формат на выбор исполнителя)
  - Аргументы командной строки
- Открывает указанный сетевой интерфейс в режим прослушивания
- Выводит в консоль (каждый пакет - отдельная строка) информацию о полученном трафике в виде:
  - IP адрес отправителя
  - Порт отправителя
  - IP адрес получателя
  - Порт получателя
  - Номер сетевого протокола
  - Размер пакета в килобайтах
- Позволяет фильтровать трафик по:
  - IP-адресу источника трафика
  - Номеру сетевого протокола
  - Сетевому порту назначения

Дополнительный блок:
- Фильтрация должна поддерживать множественный выбор параметров, например:
  - Для IP1 и IP2 с сетевыми портами назначения 80 и 8080
  - Весь трафик протокола UDP с портом назначения 53
- Работает с протоколом IPv6

Требования к оформлению:
- Результат тестового задания должен быть в виде исходного кода, размещенного в публичном репозиториий github или gitlab
- Репозиторий должен содержать файл readme.md с инструкцией по сборке и запуску приложения
